# Security Policy

## Supported Versions

We currently support the latest version of AIT.

| Version | Supported |
|--------|-----------|
| Latest | ✅ |

## Reporting a Vulnerability

If you discover a security vulnerability within AIT, please report it responsibly.

Please include:

- Description of the vulnerability
- Steps to reproduce
- Possible impact
- Suggested fix (if available)

Do NOT create a public issue for security vulnerabilities.

Instead contact:

varunteja.chunduri@gmail.com

We will investigate and respond as soon as possible.

## Security Practices

AIT follows these security practices:

- Dependency vulnerability scanning
- Code review for critical changes
- Secure handling of API keys
- Protection of sensitive configuration data
